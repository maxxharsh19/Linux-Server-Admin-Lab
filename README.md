# Linux Server Administration Lab

## Overview

This project demonstrates basic Linux system administration tasks performed on an Ubuntu server.

## Tasks Performed

- System Update
- User and Group Management
- File and Directory Permissions
- Ownership Management
- SSH Configuration
- Nginx Installation
- Apache Installation
- Log Analysis
- Bash Backup Script

## Technologies Used

- Ubuntu Linux
- Bash
- SSH
- Nginx
- Apache
